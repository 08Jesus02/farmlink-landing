# Assets

Coloca aquí los recursos visuales editables de la landing.

Sugerencias:

- `hero-farmlink.jpg`: imagen principal del hero.
- `logo-farmlink.svg`: logo oficial si decides reemplazar el texto `LOGO`.
- Fotos adicionales del campo, cultivos, equipos o dashboard del producto.

Si no agregas la imagen principal, la tarjeta del hero seguirá mostrando el degradado y el texto de marcador.
